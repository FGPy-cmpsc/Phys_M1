# Моделирование электростатического поля плоского конденсатора методом моментов

Этот проект рассчитывает распределение потенциала и силовых линий электрического поля между двумя прямоугольными металлическими пластинами при заданной разности потенциалов. Расчёт выполняется методом моментов в 3D, а визуализация строится для сечения `XZ`.

## 1. Установка и использование

### Шаг 1. Клонировать проект

```bash
git clone https://github.com/FGPy-cmpsc/Phys_M1.git 
cd Phys4SemPhys/M1
```

### Шаг 2. Создать и активировать виртуальное окружение

Windows PowerShell:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate
```

Linux/macOS:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### Шаг 3. Установить зависимости

```bash
uv pip install -r requirements.txt
uv pip install scipy
```

### Шаг 4. Запустить программу

Интерактивный режим:

```bash
python3 main.py
```

Запуск с готовым пресетом:

```bash
python3 main.py --preset ideal
python3 main.py --preset fringe
python3 main.py --preset asymmetric
python3 main.py --preset dielectric
```

Расчёт без GUI с сохранением картинки:

```bash
python3 main.py --headless --save-image out.png
```

Расчёт без GUI с JSON-результатом:

```bash
python3 main.py --headless --json
```

Пример headless-запуска с собственными параметрами:

```bash
python3 main.py --headless --json \
  --l1 0.10 --w1 0.10 --t1 0.003 \
  --l2 0.10 --w2 0.10 --t2 0.003 \
  --gap 0.008 --voltage 100 --n 13 --eps 1.0 \
  --save-image out.png
```

## 2. Тесты

Тесты запускают 5 отдельных headless-расчётов с разными наборами параметров.

```bash
python3 -m unittest -v test_headless_runs.py
```

При каждом запуске тестов сохраняются изображения:

```bash
test_outputs/dense_air_gap.png
test_outputs/fringing.png
test_outputs/asymmetric_plate.png
test_outputs/dielectric.png
test_outputs/low_voltage_insulation.png
```

Успешный тест означает, что:

* расчёт завершился без ошибки
* ёмкость получилась положительной
* заряды пластин имеют противоположные знаки
* изображение для сценария было сохранено

## 3. Входные параметры

Параметры можно задавать через GUI или через аргументы командной строки.

Основные параметры:

* `L1`, `W1`, `T1` - длина, ширина и толщина первой пластины
* `L2`, `W2`, `T2` - длина, ширина и толщина второй пластины
* `gap` - зазор между внутренними гранями пластин
* `voltage` - разность потенциалов между пластинами
* `n` - целевое число разбиений вдоль меньшей стороны
* `eps` - относительная диэлектрическая проницаемость среды
* `y-slice` - координата сечения для визуализации

## 4. Ограничения входных данных

Одинаковые ограничения действуют и в GUI, и в headless-режиме:

* `L1`, `L2`: от `0.020` до `0.30` м
* `W1`, `W2`: от `0.020` до `0.30` м
* `T1`, `T2`: от `0.001` до `0.02` м
* `gap`: от `0.003` до `0.12` м
* `|V|`: от `1` до `500` В
* `n_elements`: от `3` до `50`
* `epsilon_r`: от `1.0` до `10.0`
* `y-slice`: от `-0.15` до `0.15` м
* оценочное суммарное число элементов не должно превышать `3000`
* размер элемента должен быть положительным и не больше зазора
* толщина пластины должна быть существенно меньше её размеров в плоскости

## 5. Что считает программа

Программа:

* строит дискретизацию поверхностей двух пластин
* собирает матрицу влияния метода моментов
* решает систему линейных уравнений для зарядов элементов
* вычисляет ёмкость системы
* строит карту потенциала и силовые линии поля в сечении

## 6. Ограничения модели

* расчёт поля ведётся в 3D, но отображение делается в 2D-сечении
* боковые грани толщины пластины не учитываются
* среда считается однородной
* формула `C_ideal` - емкость бесконечного плоского конденсатора с данными параметрами, используется только как ориентир для сравнения
